How to run

## Backend

```bash
cd backend

# Run
npm run start:dev

# Browser url test rest api
http://localhost:8080/api/docs

# Run with docker compose
docker compose up -d

# Browser url test rest api
http://127.0.0.1:3000/api/docs

# ពេលដែល run backend ជាមួយ docker ត្រូវទៅប្តូរ path url ក្នុង Frontend connect ទៅ backend docker
 - frontend/src/lib/url

```

## Frontend

```bash
cd frontend

# Run
npm run dev
```
